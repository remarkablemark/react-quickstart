# react-quickstart

A template to get started with React.

This is only recommended for **_quick prototyping_** and **not to be used in _production_**.

### Install

```sh
$ git clone https://github.com/remarkablemark/react-quickstart.git
```

### Run

```sh
$ cd react-quickstart
$ open index.html
```
