# react-quickstart

A template to get started with React.

This is only recommended for **_quick prototyping_** and **not for _production use_**.

### Usage

```sh
$ git clone https://github.com/remarkablemark/react-quickstart.git
$ cd react-quickstart
$ open index.html
```
